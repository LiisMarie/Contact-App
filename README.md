# Core data based contact app

Written in Swift.

Contains 3 entities.

Person +---0< Contact >0---+ ContactType

Features:
* Adding, editing, deleting persons
* Adding, editing, deleting contacts
* Contacts view for certain person
* Contacts view for certain contact type
